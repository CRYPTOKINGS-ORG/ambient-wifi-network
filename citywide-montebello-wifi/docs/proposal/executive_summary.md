# Executive Summary

The City of Montebello can deploy a fully open, citywide Wi‑Fi network
providing free public access across residential neighborhoods, commercial
corridors, civic buildings, and parks. The network uses fiber, microwave,
and optional satellite backhaul, combined with rooftop distribution nodes
and pole‑mounted access points.

The SSID is:

Montebello-Free-WiFi

No password. No splash page. No login.

This proposal outlines coverage, backhaul, RF design, security, deployment
phases, and maintenance.
