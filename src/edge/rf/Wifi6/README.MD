# WiFi 6 RF Front-End (2×2 MIMO)

This module models a WiFi‑6 RF front‑end for use inside the Edge Node.

## Improvements over WiFi 5
- Higher linearity PAs for 1024‑QAM
- Lower‑noise LNAs
- Better filter insertion loss
- MU‑MIMO antenna support
- Higher isolation RF switches
- Support for 160 MHz channels

## Files
- `rf_frontend6.h` — WiFi‑6 RF model
- `rf_frontend6.cpp` — implementation placeholder
- `main.cpp` — test harness
